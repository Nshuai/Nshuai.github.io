CSS

CSS就是Cascading Style Sheet的缩写，中文译作“层叠样式表”或者是“级联样式表”，是用于控制网页外观处理并允许将网页的表现与内容分离的一种标记性语言

CSS不需要编译,可以直接由浏览器执行(属于浏览器解释型语言)，是Web网页开发技术的重要组成部分。

CSS的好处：

  • 使用CSS样式可以有效地对页面进行布局，更加灵活多样。

  • 使用CSS样式可以对页面字体、颜色、背景和其他效果实现精确控制，同时对它们的修改和控制变得更加快捷，更加强大。

  • 站点中所有的网页风格都使用一个CSS文件进行统一控制，达到一改全改。还可以快速切换主题，我们可以把HTML比作是骨架，CSS是衣服。同一个HTML骨架结构，不同CSS样式，所得到的美化布局效果不同。

  • CSS可以支持多种设备,比如手机,PDA,打印机,电视机,游戏机等。

  • CSS可以将网页的表现与结构分离，使页面载入得更快,更利于维护，这也是我们的最终目的。

CSS的语法：

CSS的基本语法由选择器、属性、属性的值组成，如果选择符有多个属性，由分号隔开。

注意，这里的代码都是英文格式，例如花括号、冒号和分号。

CSS是为了控制标签的样式，因此CSS的语法只有选择器、属性、属性的值，
选择器是为了找到相关的标签
属性和属性的值是对标签进行更改

1、CSS的引入方式

css有三种引入方式，分别是行内样式，嵌入样式以及链接式。我们需要根据不同的场合不同的需求来使用不同的样式。

  • 行内样式：写在在元素的style属性中的样式，这种样式仅限在元素的内部起作用。
    不推荐使用，不利于后期的维护

    <div style="color: white;background-color: #369;text-align: center">行内设置</div>

  • 嵌入式：嵌入式，是把CSS样式写在HTML文档内部head标签中的style标签里。浏览器加载HTML的同时就已经加载了CSS样式了。

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>css</title>

<!--    第二种引入方式-->
    <style>
        p{
            color:green;
        }
    </style>
</head>
<body>

<!--引入方式1-->
<div style="color: white;background-color: #369;text-align: center">行内设置</div>

<p>sunshine</p>

</body>
</html>

  • 链接式:把CSS样式写在HTML文档的外部，一个后缀为 .css 的外部样式表中，然后使用时在head标签中，使用link标签的href属性引入文件即可。

    当CSS样式需要应用在很多页面时，外部样式表是最理想的选择。在使用外部样式表的情况下，我们可以通过改变一个文件来改变这所有页面的外观。

    div{
      color: white;
      background-color: #369;
      text-align: center
}

    <head>
<!--在head中引用外部css文件-->
<link rel="stylesheet" href="common.css">

</head>

<body>
<div>链接式</div>
</body>

组合选择器

  • 后代选择器与子代选择器：
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

    <style>
        /*后代选择器*/
        .c1 .c2{
            color:red;
        }/*找到的是 所有c1 子代下的 c2*/
        
        
        .c1 .c2 .c3{
            color:red;
        }
        /*找到的是 所有c1子代c2 下的c3，且个子代的个数越多，越精确*/
        
        /*子代选择器*/
        /*只找c1子代下的c3，而不是c1中全部的c3*/
        .c1 > .c3{
            color:green;
        }

    </style>

</head>


<body>


<div class="c1">
    <div class="c2">
        <div class="c3">item2</div>
    </div>
</div>

<div class="c1">
    <div class="c2">item2</div>
</div>


</body>
</html>
  • 与或选择器

div.c1 即与选择器，即仅有div中的c1在被选中，且一定要是紧跟着的，不能空格，空格了的话就变成子代选择器了

，则是或选择器，即逗号前后的标签都使用相关的样式

<!DOCTYPE html>

            background-color:#336699;
            font-size:32px;

    </style>
</head>
<body>

兄弟选择器

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>兄弟选择器</title>

    <style>
        /*毗邻选择器：即只能选择与.c2标签相邻的.c3
        ,c1也是c2的毗邻，但是网页是由上到下加载的，所有不会作用到c1*/
        .c2 + .c3{
            color:red

        .c4 ~ div{

</head>

E[att]          匹配所有具有att属性的E元素，不考虑它的值。（注意：E在此处可以省略。
                比如“[cheacked]”。以下同。）   p[title] { color:#f00; }
                
E[att=val]      匹配所有att属性等于“val”的E元素   div[class=”error”] { color:#f00; }
 
E[att~=val]     匹配所有att属性具有多个空格分隔的值、其中一个值等于“val”的E元素
                td[class~=”name”] { color:#f00; }
特殊的，class可以用.val的方式表达 
       而其他的属性还是需要用[att~=val]方式


E[attr^=val]    匹配属性值以指定值开头的每个元素                    
                div[class^="test"]{background:#ffff00;}
 
E[attr$=val]    匹配属性值以指定值结尾的每个元素    div[class$="test"]{background:#ffff00;}
 
E[attr*=val]    匹配属性值中包含指定值的每个元素    div[class*="test"]{background:#ffff00;}

        .c2{
            font-size:32px;
            font-style:italic;
        }

        [class~=c2]{font-size:32px;
            font-style:italic;}
